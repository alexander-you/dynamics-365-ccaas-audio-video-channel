Audio & Video Channel — Deployment Assistant

Phase 3.5 prototype — local only · no Azure or Dynamics calls · nothing is deployed

Welcome

This assistant helps an administrator plan the Azure resources and configuration needed to deploy the custom Audio & Video channel for Dynamics 365 Contact Center. In this preview it collects your inputs and generates a deployment plan locally. It does not create, modify, or delete anything, and it does not contact Azure or Dynamics 365.

🔒 Do not paste secrets. Never enter connection strings, access keys, SAS tokens, or passwords here. Secrets belong in Azure Key Vault, not in this page.
🧪 MVP vs production hardening. This plan targets a working MVP. Production hardening (private endpoints, customer-managed keys, multi-region DR, lifecycle policies) is called out separately in the documentation and is not part of this preview.

Azure prerequisites

  • An Azure subscription you are allowed to deploy into.
  • Contributor on the target resource group (to create resources).
  • Owner or User Access Administrator to assign RBAC roles / Managed Identity.
  • Azure CLI and/or access to the Azure Portal.
  • Region availability for ACS Call Recording / transcription (validate with Microsoft).

Required permissions (summary)

  • Create resources → Contributor.
  • Assign roles to the Function App identity → Owner / User Access Administrator.
  • Import Power Platform solution → Dynamics 365 System Customizer / Admin (later phase).

Deployment inputs

Treated as a non-secret identifier. Leave blank if you prefer not to record it here.
Used in the naming convention <type>-<prefix>-<env>-<region>.
